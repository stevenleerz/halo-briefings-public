# Halo Briefings (Public Hosting)

This repo holds the latest daily morning briefing MP3 from Ferny Banks Farms,
hosted publicly so an iOS Shortcut can fetch it at 8 AM AEST each day.

The source code, credentials, and full briefing history live in a separate
private repo. This repo only ever contains:

- `briefing.mp3` — the latest briefing audio
- `briefing.txt` — the latest briefing text

Both are overwritten each morning.
